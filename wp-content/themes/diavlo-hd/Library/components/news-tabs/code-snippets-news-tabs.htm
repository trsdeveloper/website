Scroll to Top

The Scroll to Top is a simple script that is activated once the page scrolls down to a certain point. Once activated the 'scroll to top' button is displayed on the lower right side of the page.. the only things you can do with this component is remove it, change the button position, or change the button image.. everything else is taken care of for you via javascript.
